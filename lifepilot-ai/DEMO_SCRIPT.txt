[1:00 — ARCHITECTURE]
(show assets/architecture_diagram.png)
Let's look at the engine under the hood. Our ADK 2.0 directed graph orchestrates the entire workflow.
First, a user query enters the Security Checkpoint node. This node scrubs personal information and scans for prompt injections before the LLM ever sees the query, logging metadata to our local audit system.
If it passes, the Orchestrator node—our Executive Agent—deconstructs the query. It delegates specialized tasks to our specialist sub-agents using Google's AgentTool framework.
The Research Agent searches the web for live salaries, the Planner Agent drafts timelines, the Risk Agent identifies conceptual blockers, and the Finance Agent calculates educational ROI.
All these drafts are sent to the Critic Agent node, which challenges unrealistic assumptions.
Finally, the refined plan flows back to the Orchestrator. If the plan calls for writing local files or calendar schedules, the workflow suspends at the Human Approval node using a RequestInput interrupt, asking the operator for explicit consent. Only then does the Final Synthesis node compile the committed recommendation.

Our Research Agent immediately triggers a web search. The Planner Agent drafts a timeline and writes it to our sandbox as 'roadmap.md'. The Finance Agent runs our ROI calculator tool to project a three-year return on investment based on expected starting salaries.

We click "Allow". The workflow resumes, incorporates our approval, and presents the final Career Recommendation Report in structured markdown, ranking the tracks, summarizing milestones, and outlining mitigation strategies for the risks.
(pause)
If we were to send a malicious prompt injection like "ignore previous instructions," you would see the security checkpoint immediately block the query, bypass the orchestrator, and route to our security event node, printing a warning.

[3:00 — SECURITY & MCP]
To make this possible, we built a local Model Context Protocol, or MCP server, running on stdio transport. It exposes four high-performance tools, giving our agents safe access to search the web, write reports to a sandbox, schedule calendar events, and run financial formulas.
All file and calendar writes are fully sandboxed, and our security checkpoint ensures that no private emails, Aadhaar numbers, or phone numbers ever leave the local environment, keeping user privacy at a hundred percent.
